# Route List

1. run this command in root folder

   ```
   node ace list:routes
   ```

# Deploy Guide

1. install packages

   ```
   npm i
   ```

1. build

   ```
   npm run build
   ```

1. follow build output instruction
1. clone and update `.env` file on build folder from `.env.example`
1. run `yarn start` on `/build` folder
